## 수정 (main_server.js · askClaude)
1. **기본 max_tokens 1500 → 4096**.
2. **대화 호출부 3곳 1500/1600 → 8192**(일반대화·카드작업·과거설계).
3. **자동 이어가기(핵심)**: `stop_reason === 'max_tokens'` 감지 → 지금까지 답을 **프리필(assistant)**로 넣어 자동으로 이어받음. 최대 4회(초기1+3) = 사실상 ~32k까지 완결. 그래도 남으면 "이어서라고 하시면 계속"이라 정직 안내.
4. **프롬프트**: "핵심 결론 먼저 간결히, 불필요하게 늘이지 말고, 맨 마지막은 반드시 ⭐ 팀장 추천·예/아니오로 마무리".

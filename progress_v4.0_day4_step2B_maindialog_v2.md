# v4.0 Day4 · Task 1 v2 · sheet_list 추가 · 커넥터 격차 해소 리포트

## 근본 문제(팀장 확정)
- 커넥터 UI는 "🟢 연결됨" 표시하지만, 지니야 대화에는 **시트 목록 조회 도구가 없어** "내 구글 시트에 어떤 시트들이 있어?"에 *"연동이 안 잡혀 있다 / 지어내는 게 된다"*고 응답.
- = **UI 표시 vs 실제 Tool 노출 격차.** v1(어제)에서 시트 행 CRUD 5개는 붙였으나 **시트(파일) 목록 조회**가 빠져 있었음.

## 수정
- **`sheet_list` 도구 신규** — `drive.files.list(mimeType=spreadsheet)`로 회장님 드라이브의 시트 파일 목록(최신순 30개) 실제 조회. `doListSheets(ma)`.
- **도구 6개 `sheet_*` 정합:** `sheet_list`·`sheet_search`·`sheet_read`·`sheet_create`·`sheet_update`·`sheet_delete` (대표님 명명 100% 준수).
- **description 강화:** "구글 시트 목록·조회·수정 지금 사용 가능", "절대 '연동 안 됨/지어낸다' 금지".
- **`orderHandler` 라우팅 확장:** "시트 목록/어떤 시트/내 구글 시트/명단 조회·몇 명" → `sheetsCrud.runChat`.
- property 키 전부 영문(한글키 400 함정 없음).

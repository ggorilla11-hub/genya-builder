# 04. 기술 무기고 매트릭스

## 큰 원리: 4가지 동사

고급 AI 기술은 네 동사로 압축된다: 읽다(OCR/문서이해) · 보다(영상/음성/멀티모달) · 찾다(검색/RAG/벡터DB) · 하다(자동 조작/실행). 지니야빌더는 이 블록을 직업별로 레고처럼 조립한다.

## 난이도별 기술

- **하 — 기반(쉽고 안전):** CLAUDE.md(맥락·메모리), MCP·커넥터, 스킬, 깃/깃허브, 웹검색·웹읽기, 문서생성(워드/엑셀/PDF/PPT), API·환경변수
- **중 — 엔진(사업 동력):** 서브에이전트 병렬, 크론·웹훅(스케줄·이벤트), RAG·벡터DB, OCR·문서이해, 음성 STT/TTS, 멀티모달, 소셜리스닝, SNS 자동화·배포, 외부자동화(Make·Buffer·Zapier)
- **상 — 고급(강력하나 신중, 안전장치 후):** Computer Use / Browser Use / Playwright, 자율 에이전트 루프, 파인튜닝 + 안전벨트(휴먼인더루프·가드레일·샌드박스/최소권한)

## 직무별 핵심 기술 매칭 (◎ 핵심 / ○ 보조)

- 공통 기반(전 에이전트): CLAUDE.md, MCP·커넥터, 스킬, 깃, API·환경변수, 가드레일, 샌드박스
- 총괄(제니야): 서브에이전트 병렬, 크론·웹훅, 자율루프, 휴먼인더루프
- 고객발굴: 소셜리스닝, 크론·웹훅 / 웹검색, 브라우저유스, RAG
- 고객관리: MCP·커넥터(CRM/시트), 크론·웹훅 / RAG, 음성
- 마케팅: SNS 자동화·배포, 멀티모달 / 외부자동화, 브라우저유스
- 디자인: 멀티모달, 문서생성, 스킬
- 고객상담(금융집짓기) ★대표 차별화: RAG·벡터DB, 스킬, OCR, 멀티모달, 문서생성, 휴먼인더루프
- 개발(엄마손): 깃, 자율루프, CLAUDE.md, MCP, 가드레일, eval
- 법무·보안: RAG, 가드레일, 휴먼인더루프, 샌드박스
- 매출관리/재무: MCP·커넥터(시트), 크론·웹훅, 문서생성

## 외부자동화 전략

처음엔 Make.com·Buffer로 빠르게 시작, 반복·핵심 흐름부터 클로드 코드로 내재화. 처음부터 다 자체 구축하려 하면 공사장이 멈춘다.

## 중요한 균형

- 무기가 많다고 이기는 게 아니다. 하나의 무기를 제대로 쓰는 게 더 강하다.
- 대표의 진짜 무기는 기술이 아니라 "25년·2천 상담으로 어떤 직업이 뭘 필요로 하는지 아는 통찰(금융집짓기)". 기술은 그 통찰을 실어 나르는 그릇.
- 각 손을 켤 때 그 직무의 ◎ 기술부터 꽂는다.

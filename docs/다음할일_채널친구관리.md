# 다음 할 일 — ⑥ 카톡 채널 친구 관리 (콘텐츠 자동화 공장 파이프라인 6번째)

> 2026-06-14 기획 확정. **지금은 안 만든다(기억만).** 나중 별도 Day에 제작.
> 콘텐츠 자동화 공장(docs/진행기록.md "콘텐츠 자동화 공장" 항목)의 ⑥번째 파이프라인.

## 한 줄 목표

랜딩페이지 등으로 **금융집짓기 카톡 채널에 친구가 들어오면, 문의하기 전까지 제니야가 단계별로 자동 관리**(씨앗 뿌리기). 문의가 들어오면 대표님이 직접 수확.

## 핵심 원리

- **문의 전 = 제니야 자동** (관계의 씨앗을 뿌림)
- **문의 후 = 대표님** (사람이 수확 — 상담·결제 유도)
- **톤:** 대표 메모리의 관계온도 중 **채널친구 = 🌤 따뜻함(채널친구)** 톤 적용. 팔지 않고 신뢰부터.

## 단계 (친구 추가 시점 기준)

1. **1단계 — 친구 추가 즉시:** 환영 인사 + **무료 재무진단 도구** + 과정 랜딩 안내
2. **2단계 — 며칠 후:** 가치 제공 콘텐츠 (팔지 않고 신뢰 쌓기 — 예: "맞벌이인데 돈 안 모이는 이유")
3. **3단계 — 반응 보고:** 부드러운 권유 (과정 마감 임박 안내, 무료 진단 권유)
4. **4단계 — 고객이 질문/문의하면:** → **대표님이 직접 응대** (여기부터 사람 영역, 제니야는 빠짐)

## 기술 관문 (중요)

- **친구톡 자동발송**은 **카카오 비즈니스 채널 + Solapi 발신프로필(PFID) 승인**이 필요(승인 1~2일 소요, 외부 승인).
- **승인 전(반자동):** 제니야가 단계별 메시지를 **만들어 주고**, 대표님이 카카오톡에서 **직접 발송**.
- **승인 후(자동):** 제니야가 단계 타이밍에 맞춰 친구톡 자동 발송.
- 참고: 이 PFID 승인 관문은 CRM 친구톡/알림톡 자동발송(진행기록 v1.2 "아직 안 됨")과 동일한 외부 승인이다.

## 만들 때 메모

- 친구 명단·단계 진행상태를 어디에 저장할지(구글시트 전용 탭) 결정 — 휴먼인더루프·개인정보는 대표 저장공간에만(백서 06).
- 단계 자동 발송도 외부 발송이므로 **대표 승인** 후(또는 반자동)로. 콘텐츠 자동화 공장의 다른 손과 동일 원칙.

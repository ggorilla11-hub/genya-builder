# 07. 사업과 진행 순서

## 수익 구조

- 부트캠프 교육비 = 일시불(지니야 만드는 법을 배우는 값). 1기 기준 330만원대.
- 지니야 이용료 = 월 구독(그 지니야를 계속 돌리는 값). 서버·AI 비용 회수·수익화.
- 핵심: 부트캠프는 한 번 팔고 끝이 아니라, 수료생이 매달 내는 반복 매출(구독)이 따라온다.
- 구독가는 도그푸딩으로 1인당 실제 AI 비용 실측 후 마진 붙여 결정. 추측 금지.

## 상품 라인

- 기본 지니야: 팀장 직접 대화(대부분 고객).
- 프리미엄 지니야: 총괄 모드(제니야) — 에이전트 회의·모닝브리핑·협업(고급 유저·대표).

## 진행 순서 (도장깨기 전략)

1. 엄마 클로드 코드 설치 + 백서 작성(완료 단계)
2. 지니야빌더 제작 — 온보딩 → 개인별 홈 → 시스템 프롬프트 → 손 하나씩(발굴·문서 등)
3. 도그푸딩 — 오상열 대표 본인 제니야(총괄 모드)로 먼저 테스트
4. 1:1 도장깨기 — 직업별 1명씩 제작·검증. 1호 = 행정사.
5. 검증 후 빌드인퍼블릭 → 부트캠프 1기 모집(날짜는 검증 후 확정). 1기 8명 깊은 만족부터.

## 현실적 보완

- "10명 전부 완벽 제작"을 기다리면 부트캠프가 미뤄진다. → 핵심 3~4직업(행정사·설계사·세무사)만 깊게 검증하고 그 사례로 부트캠프를 열되, 나머지는 1기와 병행해 채운다.
- "완벽"의 함정: 100% 완벽 후에야 다음으로 가면 무한 공사. "핵심 기능이 돌아가면 충분"으로 기준을 잡는다. 완벽보다 "작동 + 빌드인퍼블릭 시작"이 먼저.

- 권장 첫 손 묶음: 소셜리스닝 + 묵은 CRM 재발굴 + 콘텐츠 SEO.

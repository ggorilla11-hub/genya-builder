지니야빌더 개인정보처리방침

오원트금융연구소(이하 "회사")가 운영하는 지니야빌더(이하 "서비스")는 이용자의 개인정보를 소중히 다루며, 「개인정보 보호법」 등 관련 법령과 Google API Services User Data Policy(제한적 사용 요건 포함)를 준수합니다.

핵심 원칙 — 제로 인그레스(Zero Data Ingress)
서비스는 이용자의 업무·고객 파일과 데이터를 회사 서버에 저장하지 않습니다. 모든 결과물과 자료는 이용자 본인의 Google Drive·Google Sheets에만 저장됩니다. 회사 서버는 이용자를 대신해 그때그때 읽어 화면에 보여주고 즉시 폐기(transient)하며, 세션 토큰만 서버 메모리에 임시 보관합니다(디스크·데이터베이스 저장 없음).

1. 수집하는 개인정보 항목

(1) 로그인 시 — 신원 확인

  • Google 로그인 사용 시: 이름, 이메일 주소, 프로필 기본 정보(구글 계정에서 제공하는 openid, email, profile 범위)
  • 카카오 로그인 사용 시: 이름(닉네임), 이메일 주소

(2) 온보딩(내 지니야 만들기) 시 — 이용자가 직접 입력

  • 직업, 하는 일, 주 고객, 반복 업무, 맡기고 싶은 일(기능), 꼭 지킬 철칙

위 정보는 이용자 본인의 Google Sheets에 저장되어, 이용자 맞춤형 비서(지니야)를 구성하는 데 사용됩니다.

(3) 서비스 이용 중 — 이용자가 요청·첨부한 내용

  • 지니야에게 보내는 대화·요청 내용, 첨부 파일(예: 보험 증권 사진, 고객 명단 등)
  • 이용자가 "기억해줘"라고 남긴 결정·요청 메모

(4) 선택적 Google 데이터 연결 시 — 아래 API 접근 권한

이용자가 캘린더·시트·드라이브 기능을 직접 사용할 때만 별도 동의를 거쳐 다음 범위에 접근합니다(로그인 단계에서는 요청하지 않습니다).

2. 개인정보의 수집·이용 목적

  • 회원 식별 및 로그인
  • 이용자 맞춤형 AI 비서(지니야) 생성 및 업무 지원(문서 작성·자료 검색·일정 브리핑 등)
  • 이용자가 요청한 작업의 수행 및 결과물 생성
  • 서비스 개선 및 오류 분석(개인 식별 정보를 제외한 범위)

3. Google 사용자 데이터의 사용 및 보호 (Limited Use)

서비스가 Google API를 통해 접근하는 정보의 사용은 Google API Services User Data Policy의 제한적 사용(Limited Use) 요건을 준수합니다.

  • Google 사용자 데이터는 이용자에게 서비스를 제공하는 목적으로만 사용합니다.
  • 광고 목적으로 사용하거나 데이터 중개상·정보 브로커 등 제3자에게 판매하지 않습니다.
  • AI/머신러닝 모델의 학습·개선에 사용하지 않습니다.
  • 사람이 데이터를 열람하지 않습니다. 다만 (a) 이용자의 명시적 동의가 있거나, (b) 보안 목적, (c) 법령 준수, (d) 개인을 식별할 수 없도록 익명·집계 처리한 경우는 예외입니다.

4. 개인정보의 저장 위치 및 보관 (제로 인그레스)

  • 이용자의 업무·고객 파일, 온보딩 입력값, 기억 메모, 작업 결과물은 이용자 본인의 Google Drive·Google Sheets에만 저장됩니다.
  • 파일 업로드 시 파일은 이용자 브라우저에서 본인의 Google Drive로 직접 전송되며, 회사 서버를 거치지 않습니다.
  • 회사 서버는 로그인 세션 토큰(접근·갱신 토큰)과 이메일만 메모리에 임시 보관하며, 디스크나 데이터베이스에 저장하지 않습니다. 로그아웃하거나 세션이 종료되면 삭제됩니다.
  • 회사 서버가 자체 보관하는 것은 공통 지식(공개된 보험 약관, 직업별 안내 자료 등)뿐이며, 이는 개인정보가 아닙니다.

6. 개인정보의 보유 및 파기

  • 세션 토큰: 로그아웃·세션 종료 시 즉시 파기(메모리에서 삭제).
  • 온보딩 입력값·기억·결과물: 이용자 본인의 Google 계정에 저장되므로, 이용자가 본인 Drive·Sheets에서 직접 삭제할 수 있습니다.
  • AI 처리를 위해 전송된 내용: 응답 생성 후 회사 서버에 남기지 않습니다.

7. 이용자의 권리

  • 이용자는 언제든지 자신의 개인정보를 열람·정정·삭제할 수 있습니다(본인 Google Drive·Sheets에서 직접 관리).
  • Google 데이터 연결은 언제든 해제할 수 있습니다: Google 계정 → 보안 → 타사 앱 액세스에서 지니야빌더의 접근 권한을 철회하면 됩니다.
  • 서비스 탈퇴(로그아웃) 시 서버 세션이 삭제됩니다.

8. 개인정보의 안전성 확보 조치

  • 모든 통신은 HTTPS로 암호화됩니다.
  • Google 로그인 시 회사는 이용자의 비밀번호를 수집·저장하지 않습니다(구글이 직접 인증).
  • 접근 권한은 기능 수행에 필요한 최소 범위로 제한하며, 데이터 연결은 이용자가 해당 기능을 사용할 때만 별도 동의를 받습니다.
